# 프로젝트 소개

영어를 더 자연스럽고 오래 지속 가능한 방식으로 배우기 위한 가이드입니다.

한국어판은 단계적으로 번역 중입니다. 현재는 한국어 홈, 목차, 검색 문구, 챕터별 진입 페이지를 먼저 제공합니다. 전체 내용이 필요한 경우 각 페이지의 중국어 원문 또는 영어 버전을 함께 참고하세요.

## 왜 이 가이드가 필요한가

영어 공부는 시험 점수만을 위한 작업이 아닙니다. 더 넓은 정보, 더 좋은 자료, 더 많은 사람과의 대화에 접근하는 방법입니다.

이 가이드는 단어, 듣기, 읽기, 말하기, 쓰기, AI 활용을 따로 떼어 보지 않고 하나의 학습 루프로 묶어 봅니다.
