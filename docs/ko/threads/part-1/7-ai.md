# AI로 영어 배우기

원문: [中文](#/threads/part-1/7-ai.md) | [English](#/en/threads/part-1/7-ai.md)

이 페이지는 한국어판의 챕터 진입점입니다. 전체 번역은 단계적으로 추가됩니다.

## 핵심 요약

- AI를 번역기처럼만 쓰지 말고 영어 코치, 피드백 도구, 복습 도구로 씁니다.
- 좋은 프롬프트보다 중요한 것은 반복 가능한 학습 루프입니다.
- 말하기, 쓰기, 읽기 자료 정리, 퀴즈, 플래시카드, 오답 복습을 하나의 흐름으로 묶습니다.
- 답을 대신 만들어 달라고 하기보다 내가 먼저 출력하고 AI에게 고쳐 달라고 요청합니다.

## 한국어 사용자에게

한국어 설명은 어려운 문법이나 뉘앙스를 이해할 때 유용합니다. 다만 실제 훈련은 가능한 한 영어 출력 비중을 늘리는 방향으로 설계하는 것이 좋습니다.
