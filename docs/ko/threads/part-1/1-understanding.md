# 인지편

원문: [中文](#/threads/part-1/1-understanding.md) | [English](#/en/threads/part-1/1-understanding.md)

이 페이지는 한국어판의 챕터 진입점입니다. 전체 번역은 단계적으로 추가됩니다.

## 핵심 요약

- 영어를 왜 배우는지 먼저 정리합니다.
- 입력만으로는 부족합니다. 말하기와 쓰기 같은 출력이 필요합니다.
- 죄책감으로 밀어붙이기보다 지속 가능한 속도와 환경을 만듭니다.
- 영어를 시험 과목이 아니라 실제로 쓰는 도구로 봅니다.
- 공부 감정, 루틴, 몰입 환경을 함께 설계해야 오래 갑니다.

## 한국어 사용자에게

중국어 원문은 학습 태도와 방법을 길게 설명합니다. 지금 바로 전체 내용을 읽고 싶다면 위의 원문 링크를 사용하세요. 한국어판은 이 구조를 유지하면서 자연스러운 표현으로 옮길 예정입니다.
