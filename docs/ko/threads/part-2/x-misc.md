# 기타

원문: [중국어](/threads/part-2/x-misc.md) | [English](/en/threads/part-2/x-misc.md)

이 장은 본문 챕터에 담기 어려운 학습 팁, 자료 선택, 강의, 유학, 공부 환경 같은 내용을 다룹니다.

## 핵심 요약

- 영어 공부를 오래 지속하려면 방법뿐 아니라 환경과 습관도 중요합니다.
- 좋은 자료를 고르는 능력은 학습 효율에 직접적인 영향을 줍니다.
- 강의나 도구는 목적이 분명할 때 가장 도움이 됩니다.
- 필요할 때 중국어 원문과 영어 버전을 함께 참고하면 맥락을 더 넓게 볼 수 있습니다.
